# 🚀 AI-Powered SEO Research Assistant

An end-to-end Python automation project that researches leading SEO experts using the YouTube Data API, transcript analysis, and Google Gemini AI to generate structured SEO research reports.

This project was developed as part of the **100Hires AI-Powered SEO Content Production** assignment to demonstrate API integration, AI-assisted research, automation, and Python development.

---

# 🎯 Project Objectives

- Research leading SEO experts
- Collect YouTube videos using the YouTube Data API
- Download video transcripts automatically
- Analyze transcript content
- Generate AI-powered research reports using Google Gemini
- Organize all research into structured Markdown files

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/tomgil31/100hires-portfolio.git
```

Move into the project directory:

```bash
cd 100hires-portfolio
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```env
YOUTUBE_API_KEY=YOUR_YOUTUBE_API_KEY
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

---

# ▶️ Usage

Run the complete automation pipeline:

```bash
python scripts/main.py
```

Example:

```
Enter SEO Expert Name:

Kevin Indig
```

The application automatically:

1. Creates an expert research template
2. Searches YouTube for relevant content
3. Downloads video transcripts
4. Performs transcript keyword analysis
5. Generates an AI-powered SEO research report
6. Saves all outputs as Markdown files
